Step to Run The Code:
- Make sure put the dataset in the dataset folder 
  the name of dataset folder must be: "CVIP100 Leaf Dataset" for CVIP100 dataset & "Swedish Leaf Dataset" for Swedish leaf dataset
  For Swedish Leaf Dataset. the structure will be: "Swedish Leaf Dataset" -> "leaf1" -> ".TIF" format
- In this zip, i don't put the dataset because the size of the folder will exceed the maximum size of submission
- Click the Run All
- For the CVIP100 Dataset, it will take approximately 20 - 30 minutes to output the result
- For the Swedish dataset, it will take longer time (Around 40 minutes)
